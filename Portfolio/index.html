Project 6: Weather App

The Weather app is a handy tool for checking the current weather conditions of any location. Users can simply enter a city or ZIP code, and the app makes asynchronous requests to a weather API to fetch real-time data. The weather display includes essential information such as temperature, humidity, wind speed, and a brief weather description. The app also offers the flexibility to switch between temperature units (Celsius and Fahrenheit) and, optionally, uses geolocation to provide weather information for the user's current location. With error handling and a visually appealing design, it ensures a seamless and informative weather-checking experience.
